Project: Chictopia Stylegallery
Client: Keds, Payless
Agency: Night Agency
Role: Architect / Lead Developer

View Project: http://www.keds.com/store/SiteController/keds/stylegallerypage
View Project: http://www.payless.com/store/home/stylegallery.jsp

Keds commissioned a style gallery to showcase Chichtopia users wearing Keds. Soon, Payless wanted a similar application but with numerous additions plus a version that would exist on their Facebook page.


With a few tweaks to the original code, I was able to have the same code base power all three implementations. If a feature was added or a change the API was needed, the application would only need to be compiled once and then distributed to all three live versions.

  • Coordinate with back end developer to design an API to integrate the application with a CMS to manage featured users.
  • Develop compilation configuration profiles and run time reflection so the application would behave differently based on what assets it was given and the API it was interfacing with.
